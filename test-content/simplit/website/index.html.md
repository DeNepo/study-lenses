# HTML file

header stuff & require the styles

```html
<!DOCTYPE html>
<html>

<head>
  <meta charset='utf-8'>
<link rel="stylesheet" href="./style.css">
</head>

<body>
```

make a header:

```html
  <h2 id='title'>hello</h2>
```

require the script:

```html
  <script src='./scripts/script.js'></script>
```


now close the document:

```html
</body>
</html>
```
